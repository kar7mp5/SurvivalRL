## Getting Started

This project is based on `WSL-Ubuntu22.04` and `Python3.10.12`.

### Install libraries

```sh
$ pip3 install -r requirements.txt
$ sudo apt-get install python3-tk
```

### Run the simulation

```sh
$ python3 main.py
```
